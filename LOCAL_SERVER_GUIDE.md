# 🚀 本地服务器启动指南

## 快速开始

### 1. 安装 Node.js 依赖
```bash
npm install
```

### 2. 启动本地服务器
```bash
npm start
```

### 3. 访问网站
打开浏览器访问：http://localhost:3000

## ✨ 本地服务器功能

### 📤 文件上传
- 支持拖拽上传视频和图片文件
- 自动按命名规范重命名文件
- 文件大小限制：100MB

### ✏️ 在线编辑
- 实时编辑记忆标题和描述
- 所有更改自动保存到本地数据库

### 🗑️ 删除管理
- 一键删除不需要的记忆
- 同时删除文件和数据库记录

### 💾 数据持久化
- 所有记忆信息保存在 `data/memories.json`
- 上传的文件自动分类存储
- 重启服务器后数据不丢失

## 📁 文件结构

```
本地完整版/
├── server.js                  # Node.js 服务器
├── package.json               # 依赖配置
├── local-features.js          # 本地功能脚本
├── data/                      # 数据存储
│   └── memories.json          # 记忆数据库
├── assets/
│   ├── videos/                # 视频文件存储
│   ├── photos/                # 图片文件存储
│   └── ...                    # 其他静态资源
└── uploads/                   # 临时上传目录
```

## 🌐 网络访问

### 局域网访问
1. 查看本机IP地址：`ipconfig`
2. 访问：`http://你的IP:3000`
3. 家里其他设备可通过此地址访问

### 外网访问（可选）
1. 路由器端口转发：3000端口
2. 使用内网穿透工具：frp、ngrok等
3. 动态DNS服务

## 🔧 开发模式

### 热重载开发
```bash
npm run dev
```

### 仅静态文件
```bash
npm run static
```

## 🎯 双模式说明

- **GitHub版本**：纯静态，适合展示和分享
- **本地版本**：全功能，适合个人使用和管理

两个版本共享相同的UI和基础功能，本地版本额外提供管理功能。

## 🚨 注意事项

1. **数据备份**：定期备份 `data/` 目录
2. **文件安全**：大文件建议压缩后上传
3. **网络安全**：外网访问时注意设置访问控制
4. **性能优化**：定期清理不需要的文件
